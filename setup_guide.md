# 🌐 دليل إعداد نظام ترجمة الملفات الاحترافي

## ✅ لا يحتاج أي API Key أو حساب Google Cloud

هذا النظام يستخدم مكتبات مجانية بالكامل:

- **deep_translator** — ترجمة مجانية بدون حدود
- **easyocr** — استخراج النص من الصور بالذكاء الاصطناعي
- **python-docx** — قراءة وكتابة ملفات Word
- **pdfplumber** — استخراج النص من PDF

---

## 📦 تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

أو تثبيت المكتبات يدوياً:

```bash
pip install deep-translator python-docx easyocr pdfplumber Pillow numpy
```

> ⚠️ **ملاحظة**: مكتبة `easyocr` تحتاج تحميل نموذج OCR عند أول استخدام (~100MB).
> سيتم التحميل تلقائياً عند أول ترجمة صورة.

---

## 🚀 تشغيل البرنامج

```bash
streamlit run app.py
```

ثم اذهب إلى **"ترجمة السير الذاتية"** من القائمة الجانبية.

## 🟢 إعداد الواتساب (طريقة الباركود)

يدعم النظام الآن الربط المباشر مع واتساب جوالك الشخصي عبر الباركود:

1. تأكد من تثبيت المتطلبات الجديدة: `pip install -r requirements.txt`
2. في ملف `.env` اجعل `WHATSAPP_MODE=web`.
3. شغل البرنامج واذهب إلى قسم **"ربط الواتساب"**.
4. سيظهر لك باركود QR، امسحه بجوالك كما تفعل في (واتساب ويب).

## 📐 التوسع المستقبلي

المحرك مبني بنمط **Handler Pattern** قابل للتوسع. لإضافة دعم صيغة جديدة:

1. أنشئ كلاس جديد يرث من `BaseTranslator`
2. حدد `SUPPORTED_EXTENSIONS`
3. اكتب دالة `translate()`
4. أضف الكلاس في `FileTranslator._handlers`

مثال لإضافة دعم `.pptx`:

```python
class PptxTranslator(BaseTranslator):
    SUPPORTED_EXTENSIONS = [".pptx"]
    
    def translate(self, file_bytes, filename, progress_callback=None):
        # كود الترجمة هنا
        pass
```
